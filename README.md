Aplikacja do powtarzania pytań z części teoretycznej egzaminu INF.03

## Funkcjonalności

- pełna baza danych wszystkich pytań, które kiedykolwiek pojawiły się na teorii
- śledzenie postępów (średni wynik, ilość przerobionych pytań, historia)
- powtarzanie niezaliczonych pytań
- wspomaganie AI do wyjaśniania pytań

Żeby działało AI należy wstawić token Mistral AI do pliku `local.properties`:
```
API_KEY=<twój klucz>
```
